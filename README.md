# Lynn's US Gym

Website for Lynn's US Gym — Corpus Christi's trusted fitness destination since 1986.

## Tech Stack

- **React** + **Vite** for fast, modern frontend
- **Tailwind CSS v4** for utility-first styling
- **React Router** for client-side navigation

## Getting Started

```bash
npm install
npm run dev
```

## Pages

- **Home** — Hero section, social proof, 3-step getting started, testimonials
- **Our Gym** — Equipment listings, amenities, facility details
- **Membership** — Pricing cards (Individual, Couple, Family)
- **Training** — Trainer profiles, session packages, booking CTA
- **Our Story** — 40-year heritage timeline and values
- **Contact** — Contact form, Google Maps embed, hours, phone
- **Free Trial** — Lead capture form with zero-commitment signup
- **FAQ** — Accordion-style answers to common questions

## Build

```bash
npm run build
```

Output goes to `dist/`.
